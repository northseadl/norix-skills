# 页面原型库

## 选型策略

先确定页面的核心任务，再选择最匹配的原型。

## 原型 1：Dashboard（监控视图）

适用：运营监控、数据分析、管理后台首页

结构：
```
┌─────────────────────────────────┐
│ 全局筛选/时间范围                  │
├──────┬──────┬──────┬───────────┤
│ KPI  │ KPI  │ KPI  │ KPI       │
├──────┴──────┴──────┴───────────┤
│ 主图表/趋势                      │
├─────────────┬──────────────────┤
│ 下钻明细     │ 快速行动/最近活动   │
└─────────────┴──────────────────┘
```

关键设计点：
- KPI 卡片突出关键数字，带同比/环比趋势
- 图表支持时间粒度切换（小时/天/周/月）
- 数据区域支持下钻到明细
- 快速行动区减少跳转成本

### 原型 2：Workspace（工作台）

适用：编辑器、配置中心、复杂表单

结构：
```
┌──────┬──────────────────┬──────┐
│      │                  │      │
│ 导航  │   主工作区         │ 属性  │
│ /    │                  │ /    │
│ 资源  │                  │ 检查  │
│ 树   │                  │ 器    │
│      │                  │      │
├──────┴──────────────────┴──────┤
│ 状态栏 / 控制台                   │
└─────────────────────────────────┘
```

关键设计点：
- 三栏布局，侧栏可折叠以最大化工作区
- 支持面板拖拽调整宽度
- 工具栏紧贴工作区顶部
- 状态栏显示上下文信息（保存状态、选中数量等）

### 原型 3：Flow Wizard（引导流程）

适用：注册、下单、配置向导等分步流程

结构：
```
┌─────────────────────────────────┐
│ 步骤导航 (1 → 2 → 3 → ✓)        │
├─────────────────────────────────┤
│                                 │
│         当前步骤内容               │
│                                 │
├─────────────────────────────────┤
│ [上一步]              [下一步]    │
└─────────────────────────────────┘
```

关键设计点：
- 步骤指示器显示当前进度和完成状态
- 支持步骤间数据回看（不破坏已填内容）
- 最终步骤为摘要确认页
- 高风险流程在最终步增加"确认声明"

### 原型 4：Feed / Timeline（信息流）

适用：社交动态、通知中心、活动日志

结构：
```
┌─────────────────────────────────┐
│ 筛选 / 分类标签                    │
├─────────────────────────────────┤
│ ┌─ 卡片/项目 ──────────────────┐ │
│ └──────────────────────────────┘ │
│ ┌─ 卡片/项目 ──────────────────┐ │
│ └──────────────────────────────┘ │
│ ┌─ 卡片/项目 ──────────────────┐ │
│ └──────────────────────────────┘ │
│        [加载更多 / 无限滚动]       │
└─────────────────────────────────┘
```

关键设计点：
- 支持虚拟化列表（大数据量场景）
- 筛选/排序不触发全页刷新
- 新内容到达时的非侵入式提示
- 滚动位置恢复（返回列表时不丢失位置）

### 原型 5：Settings / Config（设置页）

适用：用户设置、系统配置

结构：
```
┌──────┬──────────────────────────┐
│      │  分区标题                  │
│ 侧导航│  ┌────────────────────┐  │
│      │  │ 表单区域              │  │
│ ○ 通用│  │                    │  │
│ ○ 安全│  │                    │  │
│ ○ 通知│  └────────────────────┘  │
│ ○ 高级│  [保存] [重置]            │
└──────┴──────────────────────────┘
```

关键设计点：
- 侧边导航锚定各设置分区
- 表单变更即时标记为 dirty state
- 保存反馈明确（成功/失败/部分成功）
- 危险操作（删除账号等）视觉上明确区分并需二次确认

## 选型决策树

```
你的页面核心任务是什么？

├─ 监控数据和指标 → Dashboard
├─ 复杂的创建或编辑操作 → Workspace
├─ 分步完成一个流程 → Flow Wizard
├─ 浏览时间线/动态内容 → Feed / Timeline
├─ 调整配置或偏好 → Settings / Config
└─ 纯内容展示 → 根据内容量选择 Article / Landing
```
