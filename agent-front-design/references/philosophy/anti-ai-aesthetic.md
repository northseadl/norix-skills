# 反 AI 审美协议

## 为什么需要这个协议

AI 生成的 UI 存在严重的审美同质化问题。根源是训练数据的统计偏差：

1. **Indigo Bias**：Tailwind CSS 的 `bg-indigo-500` 作为默认按钮色，导致数百万网站使用紫蓝色调。AI 将"现代界面"与 indigo 色系强关联。
2. **Glassmorphism 过载**：毛玻璃效果在 2020-2023 设计作品集中过度流行，成为 AI 的"安全选择"。
3. **字体单一**：Inter、Roboto 因使用频率高而成为 AI 默认字体。
4. **抽象 3D 人物**：无面孔的几何化人物成为 AI 的通用占位图标。

结果：AI 生成的界面看起来像"换了 logo 的同一个模板"。

## 强制禁止清单

以下元素在无明确业务理由时禁止使用：

### 色彩
- ❌ Indigo/Violet 作为主色（`#4F46E5` `#6366F1` `#818CF8` 及邻近色域）
- ❌ Purple-to-Blue 渐变作为背景或按钮
- ❌ Indigo + Teal 的"科技感"双色搭配
- ❌ 大面积发光/霓虹效果

### 视觉效果
- ❌ 大面积 Glassmorphism（小范围毛玻璃用于功能性覆盖层可酌情使用）
- ❌ 抽象 3D 人物插画
- ❌ 全屏渐变 Mesh Background
- ❌ 无功能性的悬浮 Blob 装饰
- ❌ 过度的阴影层叠（shadow stacking > 2 层）

### 字体
- ❌ Inter 作为唯一/默认字体选择（使用时必须说明选择理由）
- ❌ 无差异化的 Roboto / Open Sans 搭配

### 布局
- ❌ 千篇一律的 "Hero + 3-Column Features + Testimonials + CTA" 落地页结构
- ❌ 纯对称布局（需要有意识的层次打破）

## 推荐替代策略

### 色彩策略
使用 OKLCH 色彩空间构建品牌色板，确保色系的独特性：

```
思路：从品牌 DNA 出发，而非从"看起来科技"出发

1. 锚定品牌核心色（来自 logo/品牌手册/业务调性）
2. 用 OKLCH 生成同色相的亮度阶梯
3. 选择对比色或类比色作为辅助色
4. 验证所有色彩组合通过 WCAG 2.2 AA 对比度
```

替代方案示例（非 indigo 的现代色系）：
- **Warm Neutral** — 基于暖灰 + 琥珀强调色，适合内容/编辑类产品
- **Earth Tone** — 基于绿/棕，适合健康/自然/可持续品牌
- **Deep Contrast** — 深色底 + 单一鲜明强调色（非紫蓝），适合工具/效率类产品
- **Analog Warmth** — 基于珊瑚/杏色/暖橘，适合社交/生活方式产品
- **Ink & Paper** — 近乎纯黑白 + 微妙纸质纹理，适合阅读/文档类产品

### 字体策略

根据内容调性选择字体，而非默认：

| 产品调性 | 推荐方向 | 示例 |
|---------|---------|------|
| 专业/工具 | 等宽或几何无衬线 | JetBrains Mono, Space Grotesk, Geist |
| 内容/编辑 | 衬线 + 无衬线搭配 | Playfair Display + Source Sans, Merriweather + Lato |
| 创意/时尚 | 表现力强的字体 | Outfit, Syne, Cabinet Grotesk |
| 温暖/亲和 | 圆润人文主义字体 | Nunito, Quicksand, Plus Jakarta Sans |
| 极简/克制 | 中性但有辨识度 | Satoshi, General Sans, Switzer |

### 视觉质感策略

用真实质感替代通用特效：

- **纸质纹理** — 细微的 noise overlay 代替平面色块
- **真实摄影** — 高质量场景图代替抽象 3D
- **品牌插画** — 定制风格的图标/插画代替通用素材
- **有机形状** — 自然曲线代替几何 blob
- **留白呼吸** — 充足的间距代替视觉填充

## 审美偏离度自检

每次输出设计方案时，必须回答以下自检问题：

1. **色彩独特性**：主色板是否与 Indigo/Violet 色域有 > 60° 的色相差异？
2. **字体辨识度**：字体选择是否有明确的品牌调性理由？
3. **布局差异化**：页面结构是否有至少 1 个非对称/非标准元素？
4. **质感真实性**：视觉质感是否来源于品牌 DNA 而非通用趋势？
5. **整体辨识度**：遮住 logo 后，能否通过视觉语言识别出品牌？

输出时以简短声明说明偏离度评估结果。
