# Production 模式 Playbook

## 适用时机

1. 用户要求“最终方案”
2. 用户要求“可交付方案”
3. 用户要求“上线评审/发布建议”

## 执行步骤

1. 从 Explore 结果选择唯一主方案
2. 保留最多 2 个备选并说明拒绝原因
3. 输出完整方案规格：
   - IA
   - 交互流
   - 组件树
   - 状态与数据边界
4. 运行七维评分与阻断判定
5. 输出发布建议与缓解动作

## 交付规则

1. 未完成评分不得输出结论
2. 存在 `P0/P1` 不得输出“可发布”
3. 任一维度 < 2 不得输出“可发布”
